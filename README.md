Computer Engineering student at **NUST (SEECS)**, drawn to the space between
hardware and software: designing circuits, understanding digital logic, and
learning how to make hardware respond to code. Currently building strong
foundations in software logic and circuit design, working toward embedded
systems, microcontrollers, and edge AI (TinyML).
